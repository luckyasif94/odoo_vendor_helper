Note: The module name I used is "fm_vendor_helper" instead of 'Vendor Self-Service Portal' (this point I only saw after module development)
Vendor helper installation guide:

1.copy the module to a folder
2.add the path of the folder in addons-path while running odoo
3.specify the db to be used while running, if fresh new db is used please initialise the db with "-i base" when running for the first time.
4.run odoo server
5.find the module app in apps. If not found, please use update apps list option.
6.install the module.
7.Access the vendor helper module via sales module (Vendor helper menu).
